# social-network-SocNet
Social networking system - PHP

## Authors
* Kazezova Anar
* Zhomartkali Kadyrgali

## Functionalities
1. User authentication
2. Profile
3. Post
   1. CRUD
   2. Leave comment, like, share
4. Photo
   1. Upload, delete
   2. Leave comment, like, share, tag person 
5. Search
6. Friends
7. Chat

## Technologies
* PHP
* MySQL
* HTML/CSS
* JS
